Application of Linear Consensus Protocol to Space Debris Deorbiting

Skills Used

Python, Julia, LaTeX

Overview

This is a simulation-based project with the goal of commanding autonomous robots that will deorbit debris in orbit. For our purposes, we design the control protocol that tells the robots to rendezvous with the debris they detect, and we assume the specific deorbiting mechanism is given. This is achieved by applying the Linear Consensus Protocol approach to a series of robots and debris, intelligently assigning the most optimal debris to each drone, and leaving the debris with no connections so that they move in a straight line. This project demonstrates proficiency in programming Python, Julia, LaTeX, and understanding of concepts in controls and multi-agent systems.

My Contributions

Designed the algorithm that detects the available debris, estimates which of the available robots will be closer to that debris in the near future, and assigns that robot to that debris. Also designed the algorithm to "deorbit" the debris once the robot is within a certain distance of it, as well as the function to make the robots and debris move as desired, and plotting.

This project was first written in Python, and I translated it to Julia for more efficient computing as the simulations can get quite slow as the number of objects increase.
